### Numero Especial 

* Algoritmo NumeroSpecial
	Leer n
	si n == 100 Entonces
		Imprimir 'Este es un numero especial'
	SiNo
		// diferentes de 100
		si n > 1000 & n % 10 == 0 Entonces
			Imprimir 'Este numero es casi especial'	
		SiNo
		Imprimir 'Este numero es regular'	
		FinSi
	FinSi
	
	
FinAlgoritmo
